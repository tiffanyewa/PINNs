# PINNs – Physics-Informed Neural Networks

Jupyter notebooks implementing Physics-Informed Neural Networks (PINNs) for geophysical applications.

## Contents
- `Tomo_heart.ipynb` — Eikonal PINN for seismic tomography; models forward travel-time recovery using a synthetic heart-shaped velocity model and compares PINN predictions against reference PDE solutions
- `True_velocity_model.ipynb` — construction and analysis of a true velocity model for seismic imaging
- `wavefield_PINNs.ipynb` — PINN-based wavefield modelling

All work completed independently by Tiffany Ajayi.
